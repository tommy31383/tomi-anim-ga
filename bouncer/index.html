Sheet sẽ xuất
💡 Cách import vào game engine

Unity: kéo file vào Assets, set Texture Type = Sprite (2D), Sprite Mode = Multiple, click "Sprite Editor" → Slice → Grid By Cell Size, set bằng kích cỡ tile (in trên header "Sheet sẽ xuất"). Tạo Animation: Window → Animation, drag hết frames vào, samples = 1000 / (duration / numFrames) ms.

Godot 4: import PNG, tạo SpriteFrames → New Atlas Texture → cắt theo grid. Loop on, FPS = 1000 / (duration / numFrames).

Phaser 3: this.load.spritesheet('hop', 'tomi_bounce.png', { frameWidth: TILE, frameHeight: TILE }) → scene.anims.create({ key: 'hop', frames, frameRate, repeat: -1 })

Aseprite: File → Import Sprite Sheet → grid by tile size → save as .ase với loop frame tag.
