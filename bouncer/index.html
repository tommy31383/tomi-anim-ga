Đánh dấu vị trí trên cây vũ khí (sau khi cắt)

Đuôi (cán) — gắn vào tay
Đầu (mũi) — hướng vũ khí chĩa ra

Kéo 2 chấm trực tiếp trên ảnh để định vị: cán sẽ là trục xoay khi vung, trục cán → mũi quyết định hướng vũ khí (xoay nền tự tính từ 2 điểm này).

💡 Cách import vào game engine

Unity: Texture Type = Sprite (2D), Sprite Mode = Multiple, Slice → Grid By Cell Size theo info trên header.

Godot 4: SpriteFrames → New Atlas Texture → grid. FPS = 1000 / (duration / numFrames).

Phaser 3: load.spritesheet('attack', file, { frameWidth, frameHeight })

Aseprite: File → Import Sprite Sheet → grid by tile size.

Loop modes: loop (Hop, Idle, Run, Spell, Victory, Stunned). oneshot giữ frame cuối (Hit, Die, Attack, Spawn).
